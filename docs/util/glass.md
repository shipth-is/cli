# Command: `util glass`

## Description

Apply a Liquid Glass .icon folder to a local Xcode project

## Help Output

```help
USAGE
  $ shipthis util glass PROJECT ICON [--verbose]

ARGUMENTS
  PROJECT  Path to the .xcodeproj directory
  ICON     Path to the .icon folder

FLAGS
  --verbose  Enable verbose logging

DESCRIPTION
  Apply a Liquid Glass .icon folder to a local Xcode project
```
